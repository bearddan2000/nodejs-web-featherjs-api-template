# nodejs-web-featherjs-api-template

## Description
A POC for featherjs.

## Tech stack
- koa
- typescript
- featerjs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
curl -i localhost/messages

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
